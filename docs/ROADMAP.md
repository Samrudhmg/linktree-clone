# Project Roadmap

This roadmap tracks the development progress and future plans for the Linktree Clone.

## Phase 1: MVP Core Features (Completed)
- [x] Basic profile management (display name, bio, avatar).
- [x] Link management (create, edit, delete, reorder).
- [x] Real-time preview dashboard.
- [x] Public-facing dynamic link pages.
- [x] Responsive design for mobile and desktop.
- [x] Dark/Light mode for dashboard.
- [x] Centralized theme system and Lucide icon migration.

## Phase 2: Enhanced Customization (Current)
- [ ] Custom domain support (using Next.js middleware).
- [ ] Detailed analytics (click tracking, visitor counts).
- [ ] Advanced SEO controls for each page.
- [ ] Social profile integration (Twitter, Instagram feeds).
